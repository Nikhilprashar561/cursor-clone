# Cursor Homepage Clone (HTML & CSS)

This project is a **clone of the Cursor website landing page**, built using only **HTML and CSS**.  
The goal of this project was to practice layout design and styling.

I tried to make the clone **same to same**, from the **Navbar to the Footer**.
